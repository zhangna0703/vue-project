#文件说明
data文件夹主要存放json文件，部分放置于文件夹的是需要后期ajax的文件，其它则为静态页面引用的，如问答，银行图标对应样式，城市联系json等。
```pre

├── order                     // 后期需ajax-order接口
├── repay                     // 后期需ajax-repay接口 
├── user                      // 后期需ajax-user接口
├── alertCheck.json           // 带按钮的提示-例取消下标弹框
├── alertKnow.json            // 只有标题内容的提示内容整合
├── city.json                 // 省市区联动json
├── cityData.json             // 所有城市汇总
├── dataUploan.json           // 信用-其它资料需要上传的资料分类
├── helpDatas.json            // 帮助中心文档
├── loan_detail.json          // 贷款详情内容
├── reportHelps.json          // 信用-征信帮助
├── waterBank.json            // 银行对应图标样式

```
